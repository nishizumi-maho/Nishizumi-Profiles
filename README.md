# Nishizumi-Profiles

This repository has been initialized and is ready for file uploads.

## Uploading files

You can upload files by:

1. Copying or creating files in this folder.
2. Running:

```bash
git add .
git commit -m "Add uploaded files"
git push
```

## Initial structure

- `.gitkeep` keeps the repository non-empty until you add your own files.
